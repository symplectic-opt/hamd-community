# License Summary

This file provides a plain-English summary of the HAMD Community Edition license.

It is **not** the license itself.  
The binding legal terms are in [`LICENSE`](./LICENSE).

---

## What HAMD Community Edition is licensed for

HAMD Community Edition is intended for:

- academic research
- personal experimentation
- educational use
- noncommercial technical evaluation

You may inspect the source code, run the examples, reproduce selected research results, and evaluate the method in a noncommercial setting.

---

## What you are allowed to do

You may:

- download and run the code
- modify the code for research or personal study
- use it in academic publications with appropriate citation
- evaluate whether HAMD is technically relevant to your work
- use it in classroom or educational contexts

---

## What you are **not** allowed to do

You may **not** use HAMD Community Edition for:

- commercial production workloads
- internal business operations
- revenue-generating services
- managed or hosted API offerings
- consulting deliverables for paying clients
- resale, sublicensing, or OEM distribution
- embedding the code in a commercial software product

If your use case supports a business, customer, client, or internal commercial workflow, you need a commercial license.

---

## Company use

A company may review and evaluate HAMD Community Edition internally **only** for noncommercial research/evaluation purposes.

Examples of allowed internal evaluation:
- testing the examples
- assessing whether the method is relevant
- reproducing selected benchmark behavior for technical review

Examples that require a commercial license:
- using HAMD in a portfolio workflow
- using HAMD in scheduling, routing, optimization, or planning operations
- using HAMD to support customer deliverables
- integrating HAMD into an internal production tool

---

## Publishing research with HAMD

Academic and noncommercial research use is allowed.  
If you publish work that uses HAMD, please cite the associated paper.

See [`CITATION.cff`](./CITATION.cff) or the README for citation details.

The Community Edition does **not** include:
